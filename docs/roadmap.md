# Roadmap

## Phase 1 - Repository foundation

- Project README
- MIT license
- Python package structure
- Open Data Risk Scanner prototype
- Markdown report generation
- Sample dataset metadata
- Unit tests
- GitHub Actions CI

## Phase 2 - Open Data Risk Scanner

- CKAN export support
- Metadata completeness scoring
- License and freshness checks
- Resource format assessment
- Markdown and JSON reports
- Demo reports for portfolio use

## Phase 3 - API Security Checker

Passive documentation-based checks for public API descriptions:

- HTTPS declaration
- OpenAPI availability
- authentication description
- versioning information
- documented error handling
- OWASP API Security mapping

## Phase 4 - Software Supply-Chain Radar

Passive repository review for public-sector open-source software:

- license
- dependency manifests
- CI/CD presence
- Dockerfile presence
- security policy
- SBOM readiness
- maintenance signals

## Phase 5 - Dashboard

- Local dashboard
- Risk trend view
- Exportable reports
- Portfolio screenshots
